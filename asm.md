#### 1: Viết API Products (6đ)

- Create product (name, price, desc) (1đ)
- Get all product (1đ)
- Get product theo id (1đ)
- Update product (1đ)
- Delete product (1đ)
- Validation product (1đ)
  - name is required
  - price is number & > 0

#### 2: Viết API Auth (4đ)

- SignUp (username, email, password, confirmPassword) (1đ)
- SignIn (email, password) (1đ)
- Mã hoá mật khẩu (0.5đ)
- Login tạo và về token (0.5đ)
- Validation dữ liệu đầu vào: (1đ)
  - username > 6 ký tự
  - email phải là duy nhất
  - password > 6 ký tự
  - confirmPassword trùng password
